# Release Notes

## v1.0.0 - Public repo baseline

This is the first public-facing snapshot of Learning Management Platform (LMP).

Highlights:

- Teacher, student, moderator, and admin flows are present.
- External assessments support Google Form and Microsoft Form workflows.
- Internal assessments, classroom room views, materials, library, and dashboards are included.
- Local scripts, tests, Supabase migrations, and docs are committed for collaboration.

Known follow-up areas:

- Performance tuning around batch reads and refresh boundaries.
- Further hardening of the internal assessment lifecycle.
- Continued documentation alignment as features evolve.
